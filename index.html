DCBCsite.org

Simple informational site for a group/corporation. Event calendar, staff list, contact information, blog, forms, and a video/media library.

Main User(s):

Visitors (People that want to know more about the about the group and its events)

Main Goal(s):

Landing Page Goals:
  • Have group mission statement and main information
  • Self linking header
  • Scrolling current events for group (pictures and links)
  • Simplistic horizontal navigation bar on top with intuitively named linked pages
  • "Members Only" button that pulls up a login modal
